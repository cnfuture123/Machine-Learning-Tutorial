# 绘图和可视化

## matplotlib API入门

  - matplotlib的通常引入约定是：import matplotlib.pyplot as plt.
  - Figure和Subplot：
    - matplotlib的图像都位于Figure对象中。可以用plt.figure创建一个新的Figure。
    - 不能通过空Figure绘图。必须用add_subplot创建一个或多个subplot才行。
      ```
      fig = plt.figure()
      ax1 = fig.add_subplot(2, 2, 1)
      ```
